# Enrichment Friday Workshops

Welcome to the Enrichment Friday Workshops repository! This is a collection of educational materials and lessons designed for our research lab's weekly Friday enrichment sessions.

## About

Enrichment Fridays are dedicated to learning new skills, tools, and techniques that enhance our research productivity and collaboration. Topics range from programming and scripting to productivity tools, visualization software, and best practices in research computing.

## Repository Structure

Each lesson is organized in its own directory under `lessons/` with a consistent structure:

```
lessons/
├── lesson-01-intro-to-scripting/
│   ├── README.md              # Lesson objectives and agenda
│   ├── materials/             # Slides, handouts, and reference materials
│   ├── examples/              # Working examples and demonstrations
│   ├── starter/               # Starter files for hands-on exercises
│   └── instructor_notes.md    # Notes for lesson facilitators
└── lesson-02-next-topic/
    └── ...
```

## How to Use This Repository

### For Learners
1. Navigate to the lesson you're interested in
2. Read the lesson's README.md for objectives and agenda
3. Follow along with materials during the workshop
4. Use the `starter/` files for hands-on practice
5. Refer to `examples/` for reference implementations

### For Instructors
1. Review the lesson's README.md and instructor_notes.md
2. Prepare materials from the `materials/` directory
3. Use `examples/` to demonstrate concepts
4. Distribute `starter/` files to participants

## Contributing

Have an idea for a new workshop topic? Feel free to:
1. Create a new lesson directory following the structure above
2. Include clear objectives and beginner-friendly explanations
3. Provide hands-on exercises and examples
4. Submit a pull request!
